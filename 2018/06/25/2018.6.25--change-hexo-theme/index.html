换了新主题hiker

2018-06-25 博客

新主题 hexo-theme-hiker给我的感觉相当不错。开发者的Github是iTimeTraveler。以前用的主题是Next，Next给我的感觉是很简洁，但可能是太过简洁了一点，阅读起来体验不算非常好。这款主题也是试用了几十款之后决定下来的。

选择这个主题的原因：

  • 主页可以设置背景，看起来非常大气
  • 主题颜色可以随机变换
  • 完美支持mathjax

安装

从头开始的话，先安装hexo：

然后克隆主题：

git clone https://github.com/iTimeTraveler/hexo-theme-hiker.git themes/hiker

另外如果要使用内建搜索需要安装依赖：

npm install -S hexo-generator-json-content

主题配置很简单，可参考其中文教程。

部署

至于如何部署的教程，网络上太多了，不赘述。

使用$\LaTeX$编辑公式

MathJax是一个开源JavaScript库。它支持LaTeX、MathML、AsciiMath符号，可以运行于所有流行浏览器上。
